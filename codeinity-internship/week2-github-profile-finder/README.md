# Week 3 — GitHub Profile Finder

Search any GitHub user and view their profile stats, top repositories, followers, and more — powered by the GitHub REST API.

## Features
- Search any GitHub username
- Profile stats: repos, followers, following, gists
- Top 3 repositories by stars
- Recent search history
- Light/dark theme toggle

## View it
Open `github-profile.html` in your browser.

## Files
- `github-profile.html` — main page
- `github-profile.css` — styling
- `github-profile.js` — app logic (uses the GitHub REST API)
