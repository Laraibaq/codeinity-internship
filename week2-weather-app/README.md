# Week 2 — Weather App

A glassmorphism-style weather app with real-time forecasts, sunrise/sunset data, and dynamic sky backgrounds.

## View it
Open `WeatherApp.html` in your browser.

## Files
- `WeatherApp.html` — main page
- `weathercss.css` — styling
- `script.js` — app logic
- `sky_bg.png` — background image
